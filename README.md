## Setup
1. `npm install`
2. Copy `.env.example` → `.env`, isi kredensial DB
3. Import `facultyware.sql` ke MySQL
4. `npm run dev`
5. Akses `http://localhost:3000`

## Fitur
- Semua fitur Push 1 (Dashboard)
- Pimpinan tambah/edit/hapus penugasan
- Pimpinan revisi kiriman tugas pegawai
- Export daftar penugasan ke Excel
- RestAPI penugasan pimpinan
