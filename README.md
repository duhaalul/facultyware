PJ: Syifa Khalishah Nugroho - 2411522028
Kumulatif: Push 1 + Push 2 + Push 3 (Submission Pegawai)

## Setup
1. `npm install`
2. Copy `.env.example` → `.env`, isi kredensial DB
3. Import `facultyware.sql` ke MySQL
4. `npm run dev`
5. Akses `http://localhost:3000`

## Fitur
- Semua fitur Push 1 & 2
- Pegawai lihat daftar tugas + status
- Pegawai submit hasil kerja + lampiran
- Pegawai edit submission saat direvisi
- Pimpinan validasi/setujui kiriman
- Export PDF bukti penyerahan tugas
- RestAPI submission & tracking status
